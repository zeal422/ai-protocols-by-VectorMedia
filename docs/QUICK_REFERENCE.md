## üîÑ Common Workflows

### The "New Feature" Flow
1. `FULLINDEX` (Map codebase)
2. `ULTRATHINK` (Plan architecture)
3. `ANTI-GENERIC` (Implement UI/Logic)
4. `FULLSPEC` (Add tests)
5. `COMPREHENSIVE` (Final review)

### The "Bug Fix" Flow
1. `DEEPDIVE` (Isolate root cause)
2. `AUTODEBUG` (Apply fix)
3. `FULLSPEC` (Add regression test)
4. `BESTPRACTICES` (Check for similar issues)

### The "Security Audit" Flow
1. `SECAUDIT` (Identify vulnerabilities)
2. `BIGPAPPA` (Check logic flaws)
3. `SAFEREFACTOR` (Apply fixes)
4. `GITFLOW` (Prepare secure PR)
